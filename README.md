# CS-260 Course Project  

CS-260 is a comprehensive course project repository containing multiple labs, assignments, and bonus tasks designed for software development and computer science education. The project includes Java-based implementations for various programming concepts, including inheritance, interfaces, data structures, and GUI development.  

---

## 🧩 Features & Functionality  
- **Lab1-CourseIntroduction**: Basic Java setup and environment configuration.  
- **Lab2-Inheritance**: Implementation of object-oriented inheritance principles.  
- **Lab3-UnionTest**: Testing and union operations in Java.  
- **Lab4-InterfaceDesign**: Designing and implementing Java interfaces.  
- **Lab5-SetsMapsExceptions**: Working with sets, maps, and exception handling.  
- **Lab6-TicTacToeGUI**: Graphical user interface (GUI) development for Tic-Tac-Toe.  
- **BonusAssignment**: Additional challenges for advanced programming practice.  

---

## 🚀 Installation Instructions  

### Java Development Environment  
1. Install **Java Development Kit (JDK)** 17 or higher.  
2. Set `JAVA_HOME` environment variable to point to your JDK installation.  
3. Verify installation:  
   ```bash  
   java -version  
   javac -version  
   ```  

### Xcode Setup (for iOS/macOS Components)  
1. Install **Xcode 14+** from the Mac App Store.  
2. Ensure **Swift 5.9+** is selected in Xcode preferences.  
3. For iOS/macOS deployment:  
   - **Minimum Deployment Target**: iOS 15.0 / macOS 12.0  
   - **Swift Package Manager (SPM)**: Use `swift package resolve` to fetch dependencies.  
   - **CocoaPods**: Run `pod install` for third-party libraries (if applicable).

- **src/**: Core Java source code for labs and assignments.  
- **LabX/**: Each lab contains its own source files and configuration.  
- **SubmissionFolder/**: Contains submission artifacts and diagrams.
